## About This Research

This dissertation presents a Systematic Mapping Study (SMS) investigating the impact of AI-driven tools and techniques on collaborative software development practices. Drawing from 55 peer-reviewed publications, the study examines how tools such as GitHub Copilot, Amazon Q Developer, and ChatGPT influence team productivity, coordination, communication, and software quality in distributed development environments.

---

## Research Questions

- **RQ1 (Primary):** What is the impact of AI-driven tools and techniques 
  on collaborative software development practices?
- **RQ1.1:** How do AI-assisted pair programming tools influence 
  productivity and code quality?
- **RQ1.2:** What role do AI systems play in collaborative debugging 
  and conflict resolution?
- **RQ1.3:** How effective are ML models in assessing software quality?
- **RQ1.4:** What gaps exist in AI tool adoption and how can they 
  be addressed?

---

## Key Findings

- GitHub Copilot and Amazon Q Developer are the most widely studied AI tools in collaborative software contexts
- AI tools improve individual productivity but team-level impact remains underexplored
- Significant gaps exist in longitudinal studies, ethical/legal evaluations, and team-level metrics
- Over-reliance on AI-generated code poses risks to code quality and security
